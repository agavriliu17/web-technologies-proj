Abstract

 SerP is a web application designated to manage data regarding the comercial activities of a multinational company that offers a variety of services in different locations.

Introduction

Purpose

 Provide an easy to use application for the clients to view and buy services from the company, while providing a well structured tool for the company to manage and analyse data.

User classes and characteristics

  • Admin : manages data regarding users and services, has acces to whole database, may access statistics.

  • Unauthenticated user (Guest): has access only to the landing page and the log-in/sign-up page. May create an account.

  • Authenticated user: can view the list of available services, can order a service, may manage their account and may see their past orders.

User manual

As a client user:

 On first arrival the user is accessing the landing page. Here he/she can get general information about the company, contact the company or go to the authentication page, where one can log-in to an existing account or create a new one.
 After a succesful authentication, the user is presented the home page where he/she may view, access and order any available services.

As an admin user:

 To access the admin dash-board, the user must log-in to an Admin Account. After a succesful authorization, the admin is presented the admin page. Here he/she can see detailed statistics on user activity, manage services and orders and manage his/her account.
